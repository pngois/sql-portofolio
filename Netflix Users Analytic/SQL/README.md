# Project: Netflix User Analytics

## CONTEXT

This project analyzes a Netflix user database which contains 25,000 fictional Netflix user records generated for analysis, visualization, and machine learning practice.
It includes demographic details, subscription type, watch time, and login history for each user.

## Database Structure
The final table in MySQL (netflix_users) contains the following key columns:

User_ID: Unique identifier.

Subscription_Type: Basic, Standard, or Premium.

Watch_Time_Hours: Total viewing hours (Engagement metric).

Country: User's geographic location.

Age_Group: Segmentation created via SQL for demographic analysis.
